Page metadata for the cursor-based Connection pagination pattern

```graphql
type PageInfo {
  startCursor: String!
  endCursor: String!
  hasNextPage: Boolean!
  hasPreviousPage: Boolean!
}
```


### Fields

#### [`startCursor`](#) ([`String!`](/cartesi-rollups/api/graphql/scalars/string))

Cursor pointing to the first entry of the page


#### [`endCursor`](#) ([`String!`](/cartesi-rollups/api/graphql/scalars/string))

Cursor pointing to the last entry of the page


#### [`hasNextPage`](#) ([`Boolean!`](/cartesi-rollups/api/graphql/scalars/boolean))

Indicates if there are additional entries after the end cursor


#### [`hasPreviousPage`](#) ([`Boolean!`](/cartesi-rollups/api/graphql/scalars/boolean))

Indicates if there are additional entries before the start cursor
