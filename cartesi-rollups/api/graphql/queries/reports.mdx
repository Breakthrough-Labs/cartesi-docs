Get reports with support for pagination

```graphql
reports(
  first: Int
  last: Int
  after: String
  before: String
): ReportConnection!
```


### Arguments

#### [`first`](#) ([`Int`](/cartesi-rollups/next/api/graphql/scalars/int))
Get at most the first `n` entries (forward pagination)



#### [`last`](#) ([`Int`](/cartesi-rollups/next/api/graphql/scalars/int))
Get at most the last `n` entries (backward pagination)



#### [`after`](#) ([`String`](/cartesi-rollups/next/api/graphql/scalars/string))
Get entries that come after the provided cursor (forward pagination)



#### [`before`](#) ([`String`](/cartesi-rollups/next/api/graphql/scalars/string))
Get entries that come before the provided cursor (backward pagination)



### Type

#### [`ReportConnection`](/cartesi-rollups/next/api/graphql/objects/report-connection)

Pagination result
