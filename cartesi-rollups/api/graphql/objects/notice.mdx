Informational statement that can be validated in the base layer blockchain

```graphql
type Notice {
  index: Int!
  input: Input!
  payload: String!
  proof: Proof
}
```


### Fields

#### [`index`](#) ([`Int!`](/api/graphql/scalars/int))

Notice index within the context of the input that produced it


#### [`input`](#) ([`Input!`](/api/graphql/objects/input))

Input whose processing produced the notice


#### [`payload`](#) ([`String!`](/api/graphql/scalars/string))

Notice data as a payload in Ethereum hex binary format, starting with '0x'


#### [`proof`](#) ([`Proof`](/api/graphql/objects/proof))

Proof object that allows this notice to be validated by the base layer blockchain
